Scalability Analysis of Molecular Dynamics Simulation Software

I simulated Molecular Dynamics (MD) systems on Gromacs in Linux operating system on a supercomputer. I tested and compared scalability of MPI-based parallelization of two MD software using Bash scripting. I boosted processing speed by more than 40% after switching to a new MD software which requires input file conversion accomplished by writing Tcl scripts and using a software plugin.
